# 🌦️ Weather App

A responsive Weather App built using **HTML**, **CSS**, **JavaScript**, and a **Weather API**. The application provides real-time weather information for any city, including temperature, humidity, wind speed, and current weather conditions through API integration.

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* 💧 Humidity information
* 🌬️ Wind speed updates
* ☁️ Dynamic weather icons
* 📱 Responsive design for desktop and mobile
* ⚡ Fast API-based weather retrieval
* ❌ Error handling for invalid city names

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API (e.g., OpenWeatherMap API)

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-app.git
```

2. Navigate to the project folder:

```bash
cd weather-app
```

3. Get a free API key from your weather API provider.

4. Replace the API key in `script.js`:

```javascript
const apiKey = "YOUR_API_KEY";
```

5. Open `index.html` in your browser.
